Servicios Web - API

Proyecto académico - INTEP

Buscador de Conceptos (REST, SOAP, JSON, XML)

Definiciones

  • REST: Estilo de arquitectura para servicios web basada en recursos accesibles por URL usando HTTP.
  • SOAP: Protocolo basado en XML para el intercambio estructurado de información en servicios web.
  • JSON: Formato ligero de intercambio de datos, fácil de leer y escribir, utilizado comúnmente en APIs.
  • XML: Lenguaje de marcas extensible para codificar documentos en un formato legible tanto por humanos como por máquinas.
